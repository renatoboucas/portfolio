# Projects

Renato's portfolio includes case studies focused on AI implementation, LLM/RAG architecture, data engineering, Salesforce Marketing Cloud, Data Cloud, CRM/CDP activation, consent architecture, campaign analytics, and marketing automation.

## AI Marketing Operations Assistant

This project planned an AI assistant approach for marketing operations teams. It focused on LLMs, RAG, internal knowledge search, Salesforce Marketing Cloud support, campaign workflow guidance, troubleshooting, documentation search, and operational decision support.

The project shows Renato's ability to design practical AI tools that help internal teams work faster without relying only on generic chatbots.

## RAG Knowledge Base for Customer and Internal Support

This project focused on retrieval-augmented generation, knowledge base strategy, trusted answers, document ingestion planning, embeddings, vector search concepts, permissions, and answer evaluation.

The project shows that Renato approaches RAG as a knowledge architecture problem, not simply as a chatbot interface.

## Data Engineering Pipeline for CRM/CDP Activation

This project focused on data pipelines, customer profile data, segmentation, data modeling, CRM/CDP activation, campaign readiness, and reporting-ready outputs. Tools and platforms include SQL, BigQuery, Snowflake, AWS S3, Hightouch, Salesforce, Salesforce Marketing Cloud, and Data Cloud.

The project shows Renato's ability to connect data engineering decisions to marketing, customer experience, AI readiness, and activation outcomes.

## Salesforce Data Cloud Readiness Architecture

This project focused on customer data readiness for Salesforce Data Cloud, including identity assumptions, source system ownership, consent signals, activation destinations, customer profile attributes, and phased implementation planning.

The project shows Renato's ability to connect Salesforce architecture, CRM/CDP activation, data governance, and AI readiness before platform configuration begins.

## Event Registration Data Integration Architecture

This project designed event registration data flows that connect forms, CRM records, subscriber profiles, attendance signals, campaign follow-up, and reporting-ready datasets.

The project shows Renato's ability to turn operational data from events into structured CRM, marketing automation, segmentation, and analytics workflows.

## Subscriber Profile Architecture for Segmentation

This project modeled subscriber profile data across CRM, Salesforce Marketing Cloud, preferences, engagement, and campaign history to support maintainable segmentation and automation.

The project shows Renato's ability to design reusable customer data structures that support campaign operations, Data Cloud readiness, analytics, and future AI activation.

## Salesforce Marketing Cloud Migration Architecture

This project structured a migration from Adobe Campaign Standard into Salesforce Marketing Cloud. It covered SFMC data models, Data Extensions, CloudPages, journeys, preference centers, publication lists, Automation Studio, SQL, AMPscript, and SSJS.

The project shows deep Salesforce Marketing Cloud architecture and migration planning experience.

## Preference Center and Consent Architecture

This project designed subscriber preference and consent architecture across preference centers, publication lists, subscriber status, opt-in and opt-out logic, audit logging, Data Extensions, CloudPages, AMPscript, SSJS, and SQL.

The project shows Renato's compliance-minded approach to governance, consent, and subscriber preference management.

## Campaign Analytics Automation for Victory Beyond the Cup

This project focused on campaign reporting, SFMC automation, form submissions, UTM capture, order tracking, dashboard-ready data, segmentation, and campaign analytics.

The project shows how Renato turns campaign activity into actionable reporting and decision support.
